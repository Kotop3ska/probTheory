**a) Задача на биномиальное распределение.** <br>
При анализе данных секвенирования ДНК исследователь обрабатывает 6 независимых коротких фрагментов (ридов). Вероятность того, что отдельный рид будет ошибочно сопоставлён («маппирован») к неверному участку генома, составляет $p = 0,1$. Найти закон распределения числа ошибочно сопоставленных ридов и вычислить числовые характеристики распределения.

**Решение:** <br>
Дано:<br>
	•	$n = 6$ (число ридов)<br>
	•	$p = 0.1$ (вероятность ошибки сопоставления одного рида)<br>
	•	$q = 1 - p = 0.9$<br><br>
  Пусть X – случайная величина: число ошибочно сопоставленных ридов
